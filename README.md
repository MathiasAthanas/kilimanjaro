# Kilimanjaro Workspace

This repository is now organized into three top-level application areas:

- `backend/`: existing microservices backend (NestJS + Turbo + shared packages)
- `dashboard/`: web dashboard app(s) will live here
- `mobile/`: mobile app(s) will live here

## Structure

```text
kilimanjaro/
  backend/
    services/
    shared/
  dashboard/
  mobile/
```

## Backend Quick Start

```bash
cd backend
npm install
npm run dev
```

For full backend details, see `backend/README.md`.
